# Deomali Coffee - Premium Editorial Frontend

A high-fidelity, responsive React frontend for **Deomali Coffee**, built with **Tailwind CSS**, **Framer Motion**, and **Vite**.

## 🚀 Features
- **Editorial Experience**: High-fashion typography and asymmetrical layouts.
- **Micro-Animations**: Smooth enters, scroll-reveals, and bounciness with Framer Motion.
- **Responsive Design**: Mobile-first architecture supporting all screen sizes.
- **Filterable Shop**: Robust filtering system for roasts, flavor profiles, and pricing.
- **Glassmorphism & Texture**: 5% grain overlay and backdrop-blur effects for premium feel.

## 🛠 Tech Stack
- **Framework**: React 18
- **Bundler**: Vite
- **Styling**: Tailwind CSS 3.4
- **Animation**: Framer Motion
- **Routing**: React Router 6
- **Icons**: Lucide React

## 📂 Project Structure
- `src/components`: UI components (Navbar, Footer, Hero, ProductGrid, etc.).
- `src/pages`: Main pages (Home, Shop).
- `src/data`: Mock JSON data for products.
- `src/index.css`: Design tokens and global styles.

## 🏃 How to Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:5173` in your browser.

## 🎨 Design System
- **Brand Colors**: Espresso blacks (#100403), Neon orange (#ab3600), Eggshell paper (#fbfbe2).
- **Typography**: Space Grotesk (Headlines) & Manrope (Body).
- **Core Principle**: No 1px borders. Use background color shifts and white space to define content boundaries.
